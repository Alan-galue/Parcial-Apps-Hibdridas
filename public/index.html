Dragon Ball API

Documentación de la API REST

Explora el Universo Dragon Ball

Gestiona personajes legendarios y planetas del universo más poderoso. Una API para desarrolladores que buscan la fuerza.

Información General

Base URL
http://localhost:5000

API para gestionar personajes y planetas del universo Dragon Ball

Personajes

Gestiona los personajes del universo Dragon Ball

GET /Api/Personajes

Obtiene todos los personajes

Respuesta:
Array de objetos con datos de personajes
GET /Api/Personajes/:id

Obtiene un personaje específico

Respuesta:
Objeto con datos del personaje
POST /Api/Personajes

Crea un nuevo personaje

Campos requeridos:
name, image, description, kiBase, FavoriteFood
PUT /Api/Personajes/:id

Actualiza un personaje existente

Campos opcionales:
Cualquier campo del personaje
DELETE /Api/Personajes/:id

Elimina un personaje

Respuesta:
Mensaje de confirmación

Planetas

Explora los planetas del universo Dragon Ball

GET /Api/Planetas

Obtiene todos los planetas

Respuesta:
Array de objetos con datos de planetas
GET /Api/Planetas/:id

Obtiene un planeta específico

Respuesta:
Objeto con datos del planeta
POST /Api/Planetas

Crea un nuevo planeta

Campos requeridos:
name, image, description, poblation, color
PUT /Api/Planetas/:id

Actualiza un planeta existente

Campos opcionales:
Cualquier campo del planeta
DELETE /Api/Planetas/:id

Elimina un planeta

Respuesta:
Mensaje de confirmación

Códigos de Estado

200 OK - Petición exitosa
201 Created - Recurso creado
400 Bad Request - Datos inválidos
401 Unauthorized - No autorizado
404 Not Found - No encontrado
500 Server Error - Error interno
